JAVA TODO TASKS WEB APPLICATION USING HIBERNATE

This is the Home page of this project which directs to the index.html page Task input form. which takes input from the user and by form action submit & method post it sends the data to the HTTPservlet in which by the doPost method values are fetched and then processed to store in MySQL DB by Hibernate. Html page have another button to the all tasks JSP page in which the data is fetched and displayed by hibernate query HQL. In all tasks page each task have an delet operation to perform delet operation of task, delet button sends the data through URL Rewriting to the deljsp page which performs update operation.
